# Booking-Dashboard-app# Booking Dashboard

A professional booking dashboard portfolio project built with HTML, CSS, and JavaScript.

## Features
- Add, edit, and delete bookings
- Dashboard statistics
- Search and filter bookings
- Local storage support
- CSV export
- Responsive design

## Tech Used
- HTML
- CSS
- JavaScript

## Purpose
Built as a portfolio project to demonstrate frontend development, UI design, interactivity, and data handling.
